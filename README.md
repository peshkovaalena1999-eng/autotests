
# Автотесты Cypress

## Что внутри
- Тесты для login.qa.studio
- E2E тест для Битвы Покемонов
- Логин/пароль заменены на USER_LOGIN / USER_PASSWORD

## Как использовать
1. Скопировать проект на свой компьютер
2. Создать репозиторий на GitHub
3. Загрузить все файлы (кроме node_modules)
4. Логины/пароли для покемонов менять перед запуском в USER_LOGIN / USER_PASSWORD
5. Позже можно запускать через Cypress, если установите Node.js
